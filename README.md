# Sourdough CSS buttons

Button styles for Sourdough

## Installation

Install [nodejs](http://nodejs.org) (comes with npm).

```
npm install sourdough-buttons
```

## Build

Install dependencies:
```
npm install
```

Run the build:
```
npm run build
```

Run test build:
```
npm run buildTest
```

## Browser support

* Modern Browsers
* Internet Explorer 9+
